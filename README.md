# Digital Wallet User Behavior and Lifetime Value Analysis

## Project Overview
This project analyzes digital wallet usage patterns of 641 users across four generational cohorts (Gen Z, Millennials, Gen X, Baby Boomers) using IBM SPSS.

## Tools Used
- IBM SPSS (Descriptive statistics, Chi-Square, ANOVA, Normality testing)

## Key Analyses Performed
- Descriptive statistics (frequency, mean, median, skewness, kurtosis)
- Normality testing (Kolmogorov-Smirnov & Shapiro-Wilk)
- Outlier detection using boxplots
- Chi-Square Test of Independence (age vs payment method)
- One-Way ANOVA (age vs total transactions)

## Key Findings
- No significant association between age group and preferred payment method (χ²(9) = 3.841, p = 0.922)
- No significant difference in mean total transactions across age groups (F(3, 637) = 1.959, p = 0.118)
- Digital payment behavior has largely converged across generations
